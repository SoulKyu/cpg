       ```go
       package k8s

       import (
           "context"
           apierrors "k8s.io/apimachinery/pkg/api/errors"
           metav1 "k8s.io/apimachinery/pkg/apis/meta/v1"
           "k8s.io/client-go/kubernetes"
           "go.uber.org/zap"
       )

       // RunL7Preflight verifies cluster prerequisites for L7 policy generation:
       //   - kube-system/cilium-config ConfigMap has enable-l7-proxy: "true" (VIS-04)
       //   - kube-system/cilium-envoy DaemonSet exists (Cilium >= 1.16) OR
       //     cilium-config has enable-envoy-config: "true" (Cilium 1.14-1.15 fallback) (VIS-05)
       //
       // Failures are reported as zap.Warn calls with remediation hints and DO NOT
       // return errors — pre-flight is advisory. RBAC denials (403) are also
       // warned and continued.
       //
       // Contract: invoke ONCE per cpg invocation. Each warning fires at most
       // once per call. Callers gating on --no-l7-preflight (VIS-06) MUST simply
       // skip invoking this function entirely.
       func RunL7Preflight(ctx context.Context, client kubernetes.Interface, logger *zap.Logger) {
           checkL7ProxyEnabled(ctx, client, logger)        // VIS-04 + caches enable-envoy-config result
           checkCiliumEnvoy(ctx, client, logger)            // VIS-05 (with embedded fallback to cilium-config)
       }
       ```
       Implement the two private helpers per the matrix. For the envoy fallback in case 5, the helper re-reads cilium-config (it can be cached in a struct or simply re-fetched — single ConfigMap Get is cheap). RBAC 403 detection via `apierrors.IsForbidden(err)`.

       Warning copy:

       - VIS-04 ConfigMap NotFound: `"L7 preflight: ConfigMap kube-system/cilium-config not found. L7 policy generation requires Cilium with L7 proxy enabled. Verify Cilium is installed in this cluster."`
