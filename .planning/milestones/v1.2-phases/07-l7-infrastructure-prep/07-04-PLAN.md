<objective>
Plumb the `--l7` and `--no-l7-preflight` Cobra flags through the CLI surface and into `PipelineConfig`. In Phase 7, `L7Enabled` is a NO-OP for codegen — Phase 8 wires the actual HTTP path; Phase 9 wires DNS. This plan ensures:

Per CONTEXT decision: pre-flight DOES run when `--l7` is set on generate even in Phase 7, so VIS-04/05/06 are testable end-to-end immediately.
