  <files>pkg/labels/selector.go, pkg/labels/selector_test.go</files>
  <behavior>
    - SelectLabels(["k8s:app.kubernetes.io/name=nginx", "k8s:pod-template-hash=abc"]) -> map[string]string{"app.kubernetes.io/name": "nginx"}
    - SelectLabels(["k8s:app=redis", "k8s:pod-template-hash=xyz"]) -> map[string]string{"app": "redis"}
    - SelectLabels(["k8s:team=platform", "k8s:tier=backend"]) -> map[string]string{"team": "platform", "tier": "backend"} (all labels, no priority match)
    - SelectLabels(["k8s:pod-template-hash=abc"]) -> empty map (all labels denylisted)
    - SelectLabels(["reserved:world"]) -> empty map (non-k8s labels ignored)
    - All 7 denylist labels are excluded: pod-template-hash, controller-revision-hash, statefulset.kubernetes.io/pod-name, job-name, batch.kubernetes.io/job-name, batch.kubernetes.io/controller-uid, apps.kubernetes.io/pod-index
    - WorkloadName(["k8s:app.kubernetes.io/name=nginx", ...]) -> "nginx"
    - WorkloadName(["k8s:app=redis", ...]) -> "redis"
    - WorkloadName(["k8s:team=platform", "k8s:tier=backend"]) -> deterministic string (e.g., sorted label values joined)
    - BuildEndpointSelector returns api.EndpointSelector with correct matchLabels (plain keys, no k8s: prefix)
    - BuildPeerSelector includes k8s:io.kubernetes.pod.namespace when peerNamespace differs from policy namespace
    - BuildPeerSelector omits namespace label when same namespace
  </behavior>
  <action>
Create `pkg/labels/selector.go` with:

1. `var Denylist` -- set of 7 denylisted label keys (use map[string]struct{} for O(1) lookup).

2. `func SelectLabels(endpointLabels []string) map[string]string` -- the core heuristic:
   - Parse each label with `labels.ParseLabel(raw)`
   - Filter: keep only `l.Source == labels.LabelSourceK8s` (skip reserved:, etc.)
   - Filter: exclude keys in Denylist
   - Priority: if any label has Key == "app.kubernetes.io/name", return just that one
   - Priority: else if any label has Key == "app", return just that one
   - Fallback: return all remaining labels as map[string]string
   - Edge case: if all labels are denylisted or non-k8s, return empty map

3. `func WorkloadName(endpointLabels []string) string` -- derive file/resource name:
   - If priority label found (app.kubernetes.io/name or app): return its value
   - Fallback: sort remaining label values, join with "-", truncate to 63 chars (K8s name limit)
   - If empty: return "unknown"

4. `func BuildEndpointSelector(endpointLabels []string) api.EndpointSelector` -- uses SelectLabels, then `api.NewESFromMatchRequirements(selectedLabels, nil)`. Uses plain keys (no k8s: prefix) per Research Open Question 1.

5. `func BuildPeerSelector(peerLabels []string, peerNamespace, policyNamespace string) api.EndpointSelector` -- same as BuildEndpointSelector but adds `"k8s:io.kubernetes.pod.namespace": peerNamespace` when peerNamespace != policyNamespace (locked decision from CONTEXT.md).

Write tests FIRST in `pkg/labels/selector_test.go` using testify/assert. Cover every behavior listed above. Run tests (expect RED). Then implement. Run tests (expect GREEN).
