  <behavior>
    - NewWriter(outputDir, logger) creates a Writer
    - Writer.Write(PolicyEvent) creates file at outputDir/namespace/workload.yaml
    - File contains valid YAML (yaml.Marshal of CiliumNetworkPolicy)
    - If file already exists: unmarshal existing, call MergePolicy, write merged result
    - If namespace directory does not exist: create it (os.MkdirAll)
    - Writer.Write returns error on file system failures
    - File permissions: 0644 for files, 0755 for directories
  </behavior>
  <action>
Create `pkg/output/writer.go`:

1. `type Writer struct` with `outputDir string`, `logger *zap.Logger` fields.

2. `func NewWriter(outputDir string, logger *zap.Logger) *Writer`

3. `func (w *Writer) Write(event policy.PolicyEvent) error`:
   - Compute path: `filepath.Join(w.outputDir, event.Namespace, event.Workload+".yaml")`
   - `os.MkdirAll` for namespace directory
   - Check if file exists (`os.Stat`)
   - If exists: read file, `yaml.Unmarshal` into `*ciliumv2.CiliumNetworkPolicy`, call `policy.MergePolicy(existing, event.Policy)`, marshal merged
   - If not exists: `yaml.Marshal(event.Policy)`
   - Write with `os.WriteFile(path, data, 0644)`
   - Log: `w.logger.Info("policy written", zap.String("path", path))` for new files
   - Log: `w.logger.Info("policy updated", zap.String("path", path))` for merged files

Write tests FIRST in `pkg/output/writer_test.go`:
- Use `t.TempDir()` for isolated output directories
- Test new file creation (directory + file exist, content is valid YAML)
- Test merge-on-write (write twice with different ports, verify merged content)
- Test directory creation for new namespace
- Use `testify/assert` and `testify/require`
